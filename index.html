✦ Tarot API ✦

Free image API for the Rider-Waite tarot deck.
78 cards  |  public domain  |  no key needed

made for the react hooks series on tiktok & youtube
by petaloverflow


endpoints

/cards.json   all 78 cards
/major.json   major arcana
/minor.json   minor arcana
/wands.json   wands
/cups.json    cups
/swords.json   swords
/pentacles.json   pentacles 		each card returns

{
  "name": "The Fool",
  "image": "...ar00.jpg"
}


example 

fetch("https://petaloverflow.github.io/tarot-api/major.json")
  .then(res => res.json())
  .then(cards => console.log(cards))
